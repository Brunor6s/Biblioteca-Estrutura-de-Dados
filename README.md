🚀 Sobre o Projeto

Bem-vindo(a) ao projeto Biblioteca Virtual em Estrutura de Dados! 📖💻
Este sistema foi desenvolvido com o objetivo de aplicar conceitos fundamentais de Estruturas de Dados através da criação de uma biblioteca digital capaz de gerenciar livros, usuários e empréstimos de forma eficiente ⚡

Aqui você encontrará a implementação de diversas estruturas clássicas da computação 🧠, como listas, filas, pilhas, árvores e muito mais 🌳📦

🎯 Objetivos do Projeto

✅ Praticar conceitos de Estrutura de Dados
✅ Desenvolver lógica de programação
✅ Simular o funcionamento de uma biblioteca real 📚
✅ Organizar dados de maneira eficiente 🔥
✅ Melhorar habilidades em programação em C 👨‍💻👩‍💻

🛠️ Funcionalidades

📘 Cadastro de livros
👤 Cadastro de usuários
🔍 Busca de livros por nome, autor ou código
📥 Empréstimo de livros
📤 Devolução de livros
📋 Listagem completa do acervo
⭐ Sistema de organização eficiente dos dados
💾 Salvamento de informações
🧹 Remoção de livros e usuários

🧠 Estruturas de Dados Utilizadas
📑 Lista Encadeada

Utilizada para armazenar os livros cadastrados de maneira dinâmica 🔗

📥 Fila

Responsável pelo controle de espera para empréstimos ⏳

📚 Pilha

Usada para registrar histórico de ações ou devoluções 🔄

🌳 Árvores

Aplicadas na busca rápida de livros e organização do catálogo 🚀
